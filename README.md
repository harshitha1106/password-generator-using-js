# Password Generator Using JS

A simple and secure random password generator built with HTML, CSS, and JavaScript. Generate strong, random passwords instantly with a clean and user-friendly interface.

## Features

- Generate random passwords of customizable length
- Includes uppercase letters, lowercase letters, numbers, and special characters
- Copy password to clipboard with a single click
- Clean and responsive UI
- Secure password generation using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## How to Use

1. Open the live demo or clone the repository.
2. Select your desired password length using the slider.
3. Click the "Generate Password" button.
4. Click the copy icon to copy the generated password to your clipboard.

## Installation (Local Setup)

1. Clone the repository:
   ```bash
   git clone https://github.com/harshitha1106/password-generator-using-js.git
   ```

2. Navigate to the project folder:
   ```bash
   cd password-generator-using-js
   ```

3. Open `index.html` in your browser.
